# GreenKart

## Project Overview
GreenKart is a QA automation project created to test key user flows of the GreenKart web application.

This project demonstrates automation testing skills using Java, Selenium WebDriver, Cucumber, TestNG, and Maven.

## Tools and Technologies Used
- Java
- Selenium WebDriver
- Cucumber
- TestNG
- Maven
- Eclipse IDE
- GitHub

## Test Scenarios Covered
- Login functionality
- Search product
- Add product to cart
- Verify product is added successfully
- Place order flow
- Select country
- Accept terms and conditions checkbox
